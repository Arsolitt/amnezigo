# Amnezigo

**Amnezia** + **Go** = **Amnezigo**

CLI-утилита для генерации и управления конфигурациями [AmneziaWG](https://github.com/amnezia-vpn/amneziawg) v2.0.

## Возможности

- Генерация серверных конфигураций AmneziaWG с параметрами обфускации
- Управление клиентами (добавление, удаление, список, экспорт)
- Несколько протоколов обфускации (QUIC, DNS, DTLS, STUN)
- Автоматическое назначение IP-адресов клиентам
- Генерация правил iptables для NAT и проброса
- Параметры обфускации I1-I5 генерируются для каждого клиента при экспорте
- Динамическое переключение режима client-to-client

## Установка

```bash
go install github.com/Arsolitt/amnezigo@latest
```

Или сборка из исходников:

```bash
git clone https://github.com/Arsolitt/amnezigo.git
cd amnezigo
go build -o amnezigo .
```

## Использование

### Инициализация сервера

```bash
amnezigo init --ipaddr 10.8.0.1/24
```

Параметры:
- `--ipaddr` - IP-адрес сервера с подсетью (обязательно)
- `--port` - Порт для прослушивания (по умолчанию: случайный 10000-65535)
- `--mtu` - Размер MTU (по умолчанию: 1280)
- `--dns` - DNS-серверы (по умолчанию: "1.1.1.1, 8.8.8.8")
- `--keepalive` - Интервал keepalive (по умолчанию: 25)
- `--client-to-client` - Разрешить трафик между клиентами
- `--iface` - Основной сетевой интерфейс для NAT (по умолчанию: автоопределение)
- `--iface-name` - Имя интерфейса WireGuard (по умолчанию: awg0)
- `--endpoint-v4` - IPv4 адрес эндпоинта (например, "1.2.3.4:51820")
- `--endpoint-v6` - IPv6 адрес эндпоинта (например, "[2001:db8::1]:51820")
- `--config` - Путь к файлу конфигурации (по умолчанию: awg0.conf)

### Добавление клиента

```bash
amnezigo add laptop
amnezigo add phone --ipaddr 10.8.0.50
```

### Список клиентов

```bash
amnezigo list
```

### Экспорт конфигурации клиента

```bash
# Экспорт одного клиента со случайным протоколом
amnezigo export laptop --endpoint 1.2.3.4:51820

# Экспорт с конкретным протоколом
amnezigo export laptop --protocol quic --endpoint 1.2.3.4:51820

# Экспорт всех клиентов
amnezigo export --endpoint 1.2.3.4:51820
```

Параметры:
- `--endpoint` - Адрес сервера (использует сохранённый, если не указан)
- `--protocol` - Протокол обфускации: random, quic, dns, dtls, stun (по умолчанию: random)
- `--config` - Файл конфигурации сервера (по умолчанию: awg0.conf)

### Редактирование конфигурации сервера

```bash
# Разрешить трафик между клиентами
amnezigo edit --client-to-client true

# Запретить трафик между клиентами
amnezigo edit --client-to-client false
```

Параметры:
- `--client-to-client` - Разрешить/запретить трафик между клиентами (true/false)
- `--config` - Файл конфигурации сервера (по умолчанию: awg0.conf)

### Удаление клиента

```bash
amnezigo remove laptop
```

## Параметры обфускации

AmneziaWG использует несколько параметров для обфускации трафика WireGuard:

- **Jc, Jmin, Jmax** - Параметры мусорных пакетов
- **S1-S4** - Размерные префиксы
- **H1-H4** - Диапазоны значений заголовков (хранятся как min-max в конфиге сервера, одиночное значение в конфиге клиента)
- **I1-I5** - Custom Packet Strings (CPS) генерируются для каждого клиента при экспорте на основе выбранного протокола

### Протоколы

Каждый протокол генерирует различные паттерны I1-I5:

- **quic** - трафик, похожий на QUIC
- **dns** - трафик, похожий на DNS
- **dtls** - трафик, похожий на DTLS
- **stun** - трафик, похожий на STUN
- **random** - случайный паттерн (по умолчанию)
